___
BEM STRUCTURE: `circle-button`, `circle-button__text`
___

* circle-button:
  * Container block.
  * Can either be `<button>` or `<a>`   

* circle-button__text:
  * `<span>` that wraps the button's text.
  * You can optionally put an icon class on this instead of 
    using regular text.
